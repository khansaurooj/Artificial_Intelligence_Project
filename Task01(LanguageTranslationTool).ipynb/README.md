# 🌐 Language Translation Tool — CodeAlpha AI Internship (Task 1)

This project is part of the **CodeAlpha AI Internship**.<br>
The goal is to build a simple and effective **Language Translation Tool** using Python and open-source libraries.

---

## 📁 Project Overview

The objective is to **translate text between languages** using an intuitive interface and popular translation APIs.<br>
The project demonstrates key steps: **data input, preprocessing, translation, and results display** in an interactive notebook.

---

## 🧠 Objectives

- Take user input for source text, source language, and target language
- Preprocess and validate input
- Use translation library (e.g., `googletrans`) for real-time translation
- Display translated output
- Demonstrate solution in a user-friendly Jupyter Notebook

---

## 🧩 Dataset / Input

- **User Input:** Text string to translate
- **Languages Supported:** Most major languages (English, French, German, Hindi, etc.)
- **Libraries Used:** `googletrans`, `ipywidgets` (optional for UI)

---

## ⚙️ Steps Performed

### 🪜 Step 1: Import Required Libraries
Imported essential Python libraries (`googletrans`, `ipywidgets`, etc.)

### 📂 Step 2: User Input
Notebook cells allow user to input text and select source/target languages.

### 🔍 Step 3: Preprocessing
- Ensured valid input (non-empty, string type)
- Checked language codes

### 🤖 Step 4: Translation
- Translated text using `googletrans.Translator`
- Handled errors and unsupported languages gracefully

### 📈 Step 5: Display Results
- Output translated text to user
- Optionally display confidence or detected source language
